## 问题排查

### 如果 Sessions 仍然频繁刷新

#### 检查点
1. 确认 `refreshSessions` 使用了 `useCallback`
2. 检查 `sessionsLoading` 状态是否正确
3. 验证防抖 timer 是否正确清理

#### 调试代码
```typescript
const refreshSessions = useCallback(async () => {
  console.log('refreshSessions called, loading:', sessionsLoading);
  if (sessionsLoading) {
    console.log('Skipped: already loading');
    return;
  }
  // ...
}, [sessionsLoading]);
```

---

### 如果消息不显示

#### 检查点
1. 打开浏览器开发者工具
2. 查看 Console 是否有错误
3. 检查 Network 标签，确认 API 调用成功
4. 验证 `loadSessionMessages` 是否被调用

#### 调试代码
```typescript
const loadSessionMessages = async (force = true) => {
  console.log('Loading messages, force:', force);
  const { messages } = await pi.getMessages(WORKSPACE);
  console.log('Loaded messages:', messages.length);
  agent.loadMessages(messages, { force });
};
```

---

### 如果输入框仍然遮挡

#### 检查点
1. 检查 `.chat-composer-wrap` 是否有 `position: sticky`
2. 确认 `.chat-messages` 有 `padding-bottom: 120px`
3. 验证 z-index 层级

#### 修复验证
```bash
# 在浏览器开发者工具中执行
document.querySelector('.chat-composer-wrap').style.position
// 应该返回 "sticky"

document.querySelector('.chat-messages').style.paddingBottom
// 应该返回 "120px"
```
